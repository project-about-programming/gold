# SalesFlowCLR

Учебная версия программы мониторинга продаж на **C++/CLI CLR .NET Windows Forms** и SQLite.

## Разделы

- Главная: статистика и последние продажи.
- Товары: добавить, изменить, удалить, поиск, фильтр по категории и остатку.
- Клиенты: добавить, изменить, удалить, поиск.
- Продажи: добавить, изменить, удалить, поиск, фильтр по статусу и клиенту.
- Настройки: сброс демо-данных, обновление, информация о поддержке.

## Запуск

1. Открой `SalesFlowCLR.sln` в Visual Studio.
2. Выбери `Debug | x64`.
3. Собери `Ctrl + Shift + B`.
4. Запусти `Ctrl + F5`.

База создаётся рядом с exe: `salesflow_clr.db`.

## Технологии

- C++/CLI
- CLR .NET Framework
- Windows Forms
- SQLite


## Финальная базовая версия

В этой версии исправлена структура окна: левое меню не перекрывает рабочую область. Оставлены только базовые функции: товары, клиенты, продажи, фильтры, добавление, изменение, удаление, сброс демо-данных и статистика на главной странице.
